# 🎨 Next Market — デザインシステムガイドライン (Warm Minimalism)

Next Market は、最先端の自律AI機能とエスクロー取引を直感的に伝えるために、無駄な装飾を排し、認知負荷を極限まで低減させた **「温かみのある引き算の美学：ウォーム・ミニマリズム（Warm Minimalism）」** を全画面のデザイン言語（Design Language）として採用しています。

---

## 🎨 1. カラーパレット (Color Palette)

冷たい白黒のミニマリズムではなく、人間の生活に寄り添うオーガニックで洗練されたウォームカラーを採用し、ユーザーに高い心理的安全性と信頼感（エスクロー）を提供します。

| 役割 | カラー名 | カラーコード | 視覚的意図 |
| :--- | :--- | :--- | :--- |
| **ベース背景** | アイボリーホワイト | `#faf8f5` | 生成り色。暖かく贅沢な余白を演出し、長時間の閲覧でも目が疲れません。 |
| **パネル背景** | ピュアホワイト | `#ffffff` | 情報カードやフォームを際立たせるためのクリアな白。 |
| **ブランド/プライマリ** | テラコッタレッド | `#d85b46` | レンガ色。AIエージェントの活発な交渉や生命感、情熱を表します。 |
| **セカンダリ/サクセス** | エメラルドグリーン | `#10b981` | Stripeエスクローの安全保護、取引完了、3Dスキャン合格を表す信頼の緑。 |
| **テキストプライマリ** | チャコールグレー | `#1f2937` | 黒（`#000`）のトーンを落とし、コントラスト比を最適化した読みやすいダーク。 |
| **テキストセカンダリ** | スレートグレー | `#4b5563` | 補助テキスト、カテゴリー、取引実績数などに使用する上品なグレー。 |
| **ボーダー/ライン** | クラシックウォーム | `#eadfd3` | 極細（`1px`）の繊細なラインで、画面内の区切りをうるさくなく上品に引きます。 |
| **AIホログラムネオン** | エメラルドグリーン | `#34d399` | 3D Appraisal スキャナーやシネマグラフを未来的に彩る発光色。 |

## 📐 2. タイポグラフィ ＆ タイポグラフィ階層 (Typography)

*   **基本フォント**: Sans-Serif を基本とし、システムデフォルトの角丸を美しく描画するフォントを指定。
    `font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif;`
*   **ヘッドライン (h1, h2, h3)**: 
    太字で力強いウェイト（`font-weight: 800`）を持たせ、引き算された画面の中で圧倒的な存在感（焦点）を作ります。
*   **アイブロー (Eyebrow Text)**:
    大文字、または赤の細字（`font-size: 11px`, `letter-spacing: 1.5px`）でタイトルの上部に配置し、コンテンツの分類（コンテキスト）を上品にエスコートします。

## 🏗️ 3. 形状と余白 (Shapes & Spacing)

*   **角丸の定義 (Border Radius)**:
    尖った角はユーザーに「攻撃性・不安」を与えます。Next Market では、丸みを持たせた優しいフォルムをルール化します。
    *   **カード / パネル**: `border-radius: 12px` または `16px`
    *   **ボタン / カテゴリピル / バッジ**: `border-radius: 30px` (完全なカプセル・ピル形状)
    *   **アバター**: `border-radius: 50%` (正円)
*   **贅沢な余白 (Generous Spacing)**:
    余白は「何もデザインされていない空白」ではなく、**「情報を美しく呼吸させるための最も贅沢なデザイン要素」** です。
    要素間の `gap` を `16px` や `20px` に広げ、要素の密集を防ぎます。
*   **境界ボーダー**:
    すべてのボーダーは `1px solid #eadfd3` に統一。要素を明確に分けながらも、視覚的に邪魔をしません。

## 🔮 4. インタラクション ＆ アニメーション (Micro-interactions)

ユーザーの操作に対し、滑らかなフィードバック（おもてなし）を返します。

*   **ボタンホバー時の浮き上がり**:
    `transition: all 0.2s cubic-bezier(0.4, 0, 0.2, 1);`
    ホバー時にわずかに上に移動し（`transform: translateY(-2px)`）、影をうっすら落とすことで、クリック可能であることを直感的に伝えます。
*   **背景のすりガラス効果 (Backdrop Filter)**:
    モーダルの背景（Stripe決済、AI交渉、3D立体スキャン）は、単なる半透明の黒ではなく、**`backdrop-filter: blur(8px)`** を適用することで、背後の情報を美しくぼかし、極上のラグジュアリー感を引き出します。
*   **発光エフェクト (Sparkle Glow)**:
    AI機能（シネマ動画、3Dスキャン）の実行中は、グリーンのソフトな光彩がゆっくり明滅するアニメーションを適用します。
